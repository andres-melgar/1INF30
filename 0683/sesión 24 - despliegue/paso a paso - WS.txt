glashfisg: Configuración
 -> enlace al java en asenv.bat
 	C:\Program Files\glassfish7\glassfish\config
 -> incluir en el archivo la línea
 	set AS_JAVA = C:\Program Files\Java\jdk-21

glashfisg: iniciar y parar
 -> glassfish7\glassfish\bin\asadmin start-domain
 -> verificar en http://localhost:4848
 -> glassfish7\glassfish\bin\asadmin stop-domain

 Configuración del firewall
  -> Windows Defender Firewall With Advanced Security
  -> Inbound rules
  -> New Rule
  -> Port
  -> TCP 8080
  -> Allow the conection
  -> por defecto todo check
  -> Se le pone un nombre a la regla 

Usar el servicio Web en el WA
 -> cambiar el IP en el archivo Web.config
 -> por ejemplo: <endpoint address="http://98.82.167.212:8080/SoftPubWS_Servidor/ServicioWeb"
    binding="basicHttpBinding" bindingConfiguration="ServicioWebPortBinding"
    contract="ServicioWeb.ServicioWeb" name="ServicioWebPort" />
